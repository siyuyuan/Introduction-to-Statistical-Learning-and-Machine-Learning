Speed Dating——非诚勿扰

案例背景

近年来相亲电视节目层出不穷，婚恋市场异常火爆。据统计婚恋交友市场在2016年收入近百亿收入。每个单身狗怎么才能获得异性的芳心？婚恋网站如何才能在最短的时间内帮助有婚恋需求的人找到自己心仪的对象？或许数据科学的思维可以帮我们找到这些问题的答案。

数据介绍

案例数据提供了芝加哥商学院相亲实验数据。该数据是芝加哥商学院Ray Fisman教授和Sheena Iyengar 教授2002年至2004年组织的相亲实验数据。实验的开始，组织者在该校网站上招募相亲者。志愿者需要在网站上注册，经审核后方可参与相亲活动。注册时需要留下个人信息，包括性别、年龄、种族、从事领域和兴趣。而由于数据集变量众多，请先查看案例推文什么影响因素决定“非诚勿扰”，了解数据变量。

任务七

利用rpart()函数对男性和女性分别建立决策树模型，并绘制得到的决策树； 仔细对比，尝试解读模型结果和得到的结论； 提示：可以用par(mfrow=c(1,2))将图像并列放置。

#构建分类决策树
female_tree <- rpart(formula = 决定~吸引力+真诚+智力+幽默+雄心+共同爱好, method = "class",
                     data = Female) 
male_tree <- rpart(formula = 决定~吸引力+真诚+智力+幽默+雄心+共同爱好, method = "class",
                   data = Male) 
#绘制图像，选择第二种
par(mfrow=c(1,2))
rpart.plot(female_tree, type = 2, main = "女性的决策")
rpart.plot(male_tree, type = 2, main = "男性的决策")

  解读：   
  1. 对于女性来说，吸引力如果小于6.8则拒绝，否则继续判断。如果共同爱好大于6.5，直接同意，小于3.5，直接拒绝。如在这中间，则再根据吸引力判断，小于7.3则拒绝，大于7.3则同意。
  2. 对于男性来说，吸引力小于6.3则拒绝，否则继续判断。如幽默大于6.8，则果断同意，否则再判断。即共同爱好小于4.5拒绝，否则同意。
  结论：  
  1. 通过决策树可以看出，无论是男性和女性，吸引力对其相亲决策的影响都是最大的。   
  2. 对于女性来说，她们比较注重相亲对象与自己的共同爱好，而对于男性来说，他们更关注相亲对象幽默感如何。

计算题

有工作的信息增益

\[H(D) = -\frac {9}{15}log\frac {9}{15} - \frac {6}{15}log\frac {6}{15} = 0.971\] \[H(D|A_2 = yes) = -\frac {5}{5}log\frac {5}{5} - \frac {0}{5}log\frac {0}{5} = 0\]

\[H(D|A_2 = no) = -\frac {4}{10}log\frac {4}{10} - \frac {6}{10}log\frac {6}{10} = 0.971\] \[H(D|A_2) = \frac {5}{15}*0 + \frac {10}{15}*0.971 = 0.647\] \[g(D,A_2) = H(D)-H(D|A_2)= 0.324\] 因此有工作的信息增益为0.324

有自己房子的信息增益

\[H(D) = -\frac {9}{15}log\frac {9}{15} - \frac {6}{15}log\frac {6}{15} = 0.971\] \[H(D|A_3 = yes) = -\frac {6}{6}log\frac {6}{6} - \frac {0}{6}log\frac {0}{6} = 0\] \[H(D|A_3 = no) = -\frac {3}{9}log\frac {3}{9} - \frac {6}{9}log\frac {6}{9} = 0.918\] \[H(D|A_3) = \frac {6}{15}*0 + \frac {9}{15}*0.918 = 0.5508\] \[g(D,A_3) = H(D)-H(D|A_3)= 0.42\] 因此有自己房子的信息增益为0.42

信贷情况的信息增益

\[H(D) = -\frac {9}{15}log\frac {9}{15} - \frac {6}{15}log\frac {6}{15} = 0.971\] \[H(D|A_4 = normal) = -\frac {1}{5}log\frac {1}{5} - \frac {4}{5}log\frac {4}{5} = 0.722\] \[H(D|A_4 = good) = -\frac {4}{6}log\frac {4}{6} - \frac {2}{6}log\frac {2}{6} = 0.918\] \[H(D|A_4 = verywell) = -\frac {4}{4}log\frac {4}{4} - \frac {0}{4}log\frac {0}{4} = 0\] \[H(D|A_4) = \frac {5}{15}*0.722 + \frac {6}{15}*0.918 + \frac {4}{15}*0 = 0.608\] \[g(D,A_4) = H(D)-H(D|A_4)= 0.363\] 因此信贷情况子的信息增益为0.363
